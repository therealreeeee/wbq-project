the dummies guide to making a website

Websites are created using code to tell the computer what to do. It is a little like the instructions for a Lego model telling you where the blocks need to be. Code is written line by line, using a language such as html or java. For this example, I will use html because it is a simple version and easy to work with.

{h2} {/h2} = < h2 > < /h2 > If I use an arrow inside a line of code it would all break (like a quote inside a quote) so I am going to use {} instead. At times I have also used [] to mean {} because some styles, such as font changes and background colour, need {} in the actual code.

Always remember that html and most code is case sensitive so don't use capitals in the tags if you can

The first thing you need to know is the most important bit. Open your notepad or an html editor and write !DOCTYPE html at the top. That tells the computer that you are using html in your code and it will use html when you go into your web browser to test out your website. It is a little like telling Google which language you want it to translate from.

To start, you need {html} followed by a seprate line with {head} followed by another seperate line with {body}. This is where you tell the computer information. The head is the whole page, the body is information on the section of text, such as font, styles like bold etc.

If you want to see how it is done, open this website in notepad or do control shift i on a website to see the code. The more complicated the website, the more code that is needed.

The next thingto consider is paragraphs. That is addedby using {p} your words and then followed by {/p} at the end. The code simply makes a new paragraph in the website. As you can see, if you open this using Ctrl Shift i, I have used {p} on almost every line!

There are so many weird and wonderful things you can do in html. You can make lists, bullet points, tables and add extra details.


  • Using {ul} means that it is a bullet pointed list.
  • In each section of the list you have to do {li} to show the computer it is a different item. Make sure you always close the brackets just like in the headings by using {/li}.
  1. Using {ol} will create a list that numbers the items
  2. After each list item you have to close the brackets and open a new set if you want to start a different item.

You can also do create a space in your document. By simply typing {br} as the line of code. This is a full line and doesn't need to be ended with {/br}

The code for a marquee is simply {marquee} then your text and complete the line with {/marquee}. Marquees have their own settings you can edit like this {marquee behaviour=scroll scrollamount="7"} {/marquee}

Scroll amount means how fast it goes. The higher the number, the faster it is. There are lots of diffrent behaviours for marque such as; go up the page, down the page, across from diffrent sides. The direction it comes from can be changed using {marquee behaviour=scroll direction=left} your text here then {/marquee}


styles

Your text can be made more interesting using the code {styles}. Styles can change the fonts size, colour, the background colour etc.

To make the writing blue you can use {p style="color:blue;"} the text here {/p}. The code is saying, this is a paragraph and it is blue. The same goes for any colour. Please note; the spelling of "color", if you spell it the English way in the code, it does not work.

I am red

To centre text and change colour and font, I use

{style}

body [

background-color: rgb(8, 164, 255);

text-align: center;

color: black;

]

p [ font: 15px sans-serif ;

]

{/style}

In code terms, this means I have told the computer to do a background colour, to centre the text and that my font is black, size 15 and the font style of sans-serif.
